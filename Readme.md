# Lady Bugs!

## Introduction

This is an event mod built as a foray into Stellaris Modding; it's relatively simple!

## ToDos

* Graphics
  * For Discovering the Asteroid
  * For Finishing the Event
  * Splash page for mod upload
* Writing
  * Machine Empires
  * Hive Minds
  * Psionic Empires
  * Description Rework
* Deposits
  * Food + Minerals for Machine Empires
